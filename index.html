Minha caixa de ferramentas

Ferramentas que domino e que me permitem criar soluções criativas e funcionais para meus clientes.

  • Desenvolvimento Web

    Área da tecnologia voltada à construção de sites, aplicativos, softwares, bancos de dados e quaisquer outras ferramentas que, de certa forma, constroem a internet.

  • HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Permite a construção de websites e a inserção de novos conteúdos, como imagens e vídeos, por meio dos hipertextos.

  • JavaScript é uma linguagem de programação que permite a você implementar itens complexos em páginas web. Algumas das empresas mais conhecidas que usam JavaScript incluem: Google, Facebook, Netflix, Uber.

  • CSS é a sigla para o termo em inglês Cascading Style Sheets que, traduzido para o português, significa Folha de Estilo em Cascatas. O CSS é fácil de aprender e entender e é facilmente utilizado com as linguagens de marcação HTML ou XHTML

  • Git é o sistema de controle de versão open source. Ele é usado para controlar o histórico de alterações de arquivos e principalmente de projetos de desenvolvimento de software. Ele permite mais flexibilidade no fluxo de trabalho, segurança e desempenho.
